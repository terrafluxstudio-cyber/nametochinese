# persons 主辞典 · 语种(nationality)分布

> 数据源：本地 `tasks/names.db`（= 灌入生产 Turso 的同一份），2026-06-18 查得。免得下次又直连生产库。
> ⚠️ 字段陷阱：`nationality` 是自由文本，`SELECT COUNT(DISTINCT nationality)` = **13,555 是假象**——混了组合值（`法、意`、`德;捷`）和噪声（`1900-?` 残缺生卒年）。真正干净单语种桶只有几十个，量高度集中在头部。

## 关键洞察
- **欧洲中心**：头部 12 语种（法/意/日/俄/英/德/塞/捷/西/瑞典/罗/匈）≈ 50 万条 / **74%**。
- **偏薄语种**（相对该语言真实人名体量）：
  - **韩语**几乎不在 persons（靠名人库 korean 7262 单独撑）
  - 中文 1002 / 越南 607 / 印度 1779 / 希伯来 568 —— 极薄
  - 阿拉伯 8756 —— 对整个阿语世界仍偏少

## 决策留档（别重复讨论）
这张表能告诉"哪里薄"，**但不改变"现在扩库无价值"**：薄的语种要么没安全数据源（再爬辞典=版权红线 [[longtail-namepage-copyright-line]]），要么没人查（0 用户），且 DB 行≠页面=对 SEO 零帮助。
