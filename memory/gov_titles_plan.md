# 政治机构 / 官员职位 中英双语库 —— 计划与铁律

> 子项目。改这块前必读。性质：小而精的**权威标准术语库**，零容错。成败在权威+可溯源+时效，不在条数。

## 🚫 铁律（用户明确，不可违背）

1. **只用官方翻译**。译法冲突时按来源层级取高档，**机构名称与职位职级的来源优先级不同**：
   - **机构名称**：① 机构自身官网英文名（法定自称，最高）→ ② 政府工作报告英译本 / 外交部官网（官网缺失或英文站滞后时兜底）→ ③《中国关键词》等
   - **职位 / 职级**：① 政府工作报告英译本、《中国关键词》（中央系统译法，机构官网无系统职级对照）→ ② 外交部等官网职务表述
   - **冲突一律取高档**，绝不取低档或自行拼造
2. **不抓 wiki**，一条都不抓
3. **立场/敏感一律不碰**，宁可库小不惹麻烦：台港、涉疆涉藏、争议领土、争议组织相关机构/职位 → 全部进排除清单，不入库
4. 准确性第一：凭记忆的译法一律标 `待核`，上线前对照官方源核验；机构重组会让旧译过时（卫计委→卫健委），status 标现行/历史

## 范围（双向都做）

- **中国中央机构 + 职位 → 英文**：国务院组成部门、党中央机构（无争议项）、人大政协、两高、通用职级对照
- **外国机构 + 职位 → 中文**：美/英/法/德/日/欧盟/联合国及主要国际组织核心机构职位。来源：新华社译名标准 + 各机构官方中文名

## 技术选型

- 纯静态数据（精选 ~400-500 条），打包 TS/JSON，**客户端加载 + 模糊搜索 + 分类筛选**
- 零 DB、零 API、SSG。不走 Turso（量小，省保活）

## 数据结构

```ts
{
  zh, en, abbr,                       // 中文 / 英文 / 缩写
  category: '机构' | '职位',
  side: 'cn' | 'foreign',
  organ: '党'|'政'|'人大'|'政协'|'司法'|'国际组织',
  level,                              // 职位才有：国家级正/省部级正…
  status: '现行' | '历史',
  source,                             // 出处（必填，可溯源）
  note,                              // 易错点/译法说明
}
```
职级对照单列一表（正部级=Minister-level…），跨机构通用。
