## ⚠️ 前提更新（2026-06-21）：分清渠道的「两种价值」

> **旧假设已部分推翻：** 用户从大陆回来实测——**站大陆能打开，但加载有时慢**（无 ICP→走海外节点，慢且脆弱、时好时坏）。所以不能再一刀切说「大陆平台全白费」。

**一个渠道有两种价值，必须拆开看：**

| 渠道 | SEO 外链价值（传 Google 权重） | 直接引流（真人点进来） |
|---|---|---|
| 知乎/小红书/微博 | ≈0（nofollow + 喂百度，不是我们做的 Google） | **成立**——站既然能开，爆款笔记能送真人 |
| LinkedIn/Medium（英文） | nofollow≈0；受众非中文读者 | 弱（顺带/求职用，别当主力） |

**结论（战略重排）：**
1. **社交分发整体降级**——真主战场是**站内 SEO 内容 + 内链 + 程序化页质量**（站靠 Google 搜进人，译者撞到名字 → Google "X 的中文译名"，只有排名页接得住，复利）。
2. **要做引流社交，先解决「慢」**——否则引来的人转圈 3 秒就走（给漏水桶倒水）。提速见 `project_pending_todos.md` 🔴 P0（SSR→静态/ISR）。
3. 当前真瓶颈 = 2123 页「已发现未索引」，Google 不给爬取预算。**靠刷社交 nofollow 治不好**，要靠 ① 几条真 dofollow（译者协会资源页/被采纳投稿）② 程序化页提速+减肥+内链。
4. 受众分两层：**繁中圈（台/港/新/马+海外，用 Google）** 和 **大陆圈（能开但慢，用百度，引流可、SEO 无）**——两层都在，但 SEO 红利只在繁中+Google 那层。

## Tier 1 —— Dofollow + 全球（最值钱的 SEO 外链）

### 1. GitHub awesome-lists ⭐ 最稳
- **怎么做**：搜 `awesome chinese`、`awesome translation`、`awesome nlp chinese`、中文学习/翻译资源类仓库，给合适的列表提 PR，把本站作为"译名/音译工具"加进去。
- **挂什么链**：站点首页或最对口的工具页（如 /search 辞典、/convert 引擎）。
- **要点**：dofollow、权重好、全球可见。PR 描述写清工具用途，别只贴链接。一次提 1-2 个高质量的，别群发。

### 2. 工具目录站
- **AlternativeTo**：列为某翻译/词典工具的 alternative。
- **SaaSHub / Product Hunt / 海外"在线工具导航"站**：搜 "submit online tool free"。
- **挂什么链**：首页 + 一句话定位"外文人名地名↔中文，给翻译工作者的查译名工具"。
- **要点**：Product Hunt 多 nofollow 但带真人 + 常被二次收录；选周中上线，备 3 张截图（/search、/convert、/gov-titles）。

## Tier 2 —— 译者专业社群（外链 + 精准用户）

### 3. ProZ.com ⭐（全球译者大本营，EN-ZH 译者云集）
- **怎么做**：建 profile（个人/工具介绍处放链接）；在论坛 Chinese / translation tools 板块答译名相关问题时自然带链。
- **发什么**：先纯帮忙答几个译名/工具问题攒信誉，再在相关问题里推荐工具当辅助。
- **挂什么链**：/search（查既定译名）或对应专题页。

### 4. Chinese Language Stack Exchange（chinese.stackexchange.com）⭐
- **为什么**：高权重 dofollow + 学中文/译名的精准用户。
- **怎么做**：搜 "transliteration"、"foreign name in Chinese"、"how is X translated" 类老问题，写真正有用的答案，自然带工具或对应指南页链接（你的「外文人名音译总则 /naming-rules/general」正好是答这类问题的料）。
- **规则**：答案必须自带价值，先答 2-3 个不带链接的攒信誉，否则当广告删。

### 5. 华文翻译学会 / 译者组织资源页
- **目标**：台湾翻译学学会、香港翻译学会、各地译者协会的"资源/工具/链接"页。
- **怎么做**：邮件礼貌求收录，附一句工具价值说明。这类权威站的链值高、可信度高。
