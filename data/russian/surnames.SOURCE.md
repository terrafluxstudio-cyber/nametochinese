# 俄语姓氏词典数据来源

- **文件**：`surnames.json`（4908 个姓氏 → 中文）
- **源**：Wikidata SPARQL —— 知名俄罗斯/苏联/俄罗斯帝国籍人物（`P31=Q5` 且 `P27∈{Q159,Q15180,Q34266}`，`wikibase:sitelinks≥6`，共 15975 人）的俄语(ru) + 中文(zh) 标签。zh 标签即 zh.wikipedia 条目名，遵新华社译名标准。
- **抓取**：`scripts/ru_surnames/build.py`（单条大查询，`strict=False` 容控制字符，429 退避重试）
- **定稿**：`scripts/ru_surnames/finalize.mjs`
- **性质**：俄语姓氏 ↔ 中文音译对照，事实性参考数据，非原创表达。已重新结构化。

## 处理流程
1. **对齐抽姓**：姓氏 = 俄文全名末词 / 中文末段（`·` 分隔）。含逗号格式「Фамилия, Имя」取逗号前。中文须纯汉字 2–4 字、俄文纯西里尔 ≥3 字母。
2. **频次聚合**：同姓多人 → 取最高频中文（`n` = 出现人数）。
3. **碰撞去错**：某中文若被另一更高频的姓「占用」，则丢弃低频误配（共丢 64）。例：`Смирнов→西蒙诺夫`(n=2) 被 `Симонов→西蒙诺夫`(n=6) 占用 → 丢弃。
4. **繁转简**：opencc-js tw→cn（庫→库 / 馬→马 / 維→维 / 奧→奥 / 澤→泽 …）。
5. **人工正名/补名人**：finalize.mjs 内 `CORR`，覆盖自动结果。修被碰撞误删或中文单段(无·)被跳过的高价值名人：Смирнов 斯米尔诺夫、Ленин 列宁、Достоевский 陀思妥耶夫斯基、Чайковский 柴可夫斯基、Брежнев 勃列日涅夫、Медведев 梅德韦杰夫、Горбачёв 戈尔巴乔夫、Солженицын 索尔仁尼琴 等。

## 质量
- n≥5 的 116 个高频姓人工核对基本全对（伊万诺夫/波波夫/彼得罗夫/索科洛夫/莫罗佐夫/托尔斯泰…）。
- 长尾（n=1，约 2900 个）为单一名人来源，偶有误配风险；碰撞去错已滤掉一类系统错误。
- 重抓需注意 Wikidata WDQS 限速（曾遇 1req/min）。
